\doxysubsubsection{\texorpdfstring{coup()}{coup()}}
{\footnotesize\ttfamily bool Jeu\+::coup (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



cette fonction permet de gérer les coups des joueurs en alternance. 

Elle affiche un message indiquant à quel joueur il est de jouer. Elle demande à l\textquotesingle{}utilisateur de saisir une chaîne de caractères représentant un coup valide. Elle vérifie que la saisie est valide et effectue le coup demandé en utilisant les méthodes appropriées de la classe \mbox{\hyperlink{class_jeu}{Jeu}}. Si le roi du joueur qui vient de jouer est en échec, elle affiche un message et permet à ce joueur de saisir un autre coup valide. Si l\textquotesingle{}utilisateur saisit \char`\"{}quit\char`\"{}, elle affiche un message de fin de partie avec le résultat \char`\"{}?-\/?\char`\"{} et retourne false pour arrêter le jeu. Si l\textquotesingle{}utilisateur saisit \char`\"{}draw\char`\"{}, elle affiche un message de fin de partie avec le résultat \char`\"{}1/2-\/1/2\char`\"{} et retourne false pour arrêter le jeu. Si l\textquotesingle{}utilisateur saisit \char`\"{}resign\char`\"{}, elle affiche un message de fin de partie avec le résultat approprié en fonction du joueur qui a abandonné et retourne false pour arrêter le jeu. Si l\textquotesingle{}utilisateur saisit un coup de roque petit ou grand valide, elle effectue ce coup en utilisant les méthodes appropriées de la classe \mbox{\hyperlink{class_jeu}{Jeu}}.

\begin{DoxyReturn}{Returns}
true si le coup a été joué, false si le coup n\textquotesingle{}a pas été joué (fin de partie ou saisie invalide) 
\end{DoxyReturn}
